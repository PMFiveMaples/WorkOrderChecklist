Work Order Checklist

CSR: WO: Date:
  • General Tab
  • Data Processing
  • Lettershop/Fulfillment
  • Print
  • Foreign
  • Warehouse
  • Postage
  • Inventory

General Tab

  • Name of Job Correct?
  • Drop Date Correct?
  • Release Date Filled Out?
  • Job Type Selected?
  • If Secure, Secure Box Checked?
  • If "Print In", Box Checked?
  • If Receiving Materials, Date Entered?
  • If Materials Received, Box Checked?
  • If RUSH, is it written on all WOs?
  • Is this a not before?
    • Caution card included in job bag?
    • Box checked off?
  • Are there any special comments that need to be entered? (DP by, Drop in MA, Commingle, MUST, Partnership Merge, etc.)

Postage Tab

  • Post Affix Matches Lettershop Direction?
  • NP and CRID #s Entered in Comments?
  • Post Status Correct?
    • If “Pending,” Postage Invoice Created?
      • Sent to Client?
      • Put in Job Bag
    • If “Ship,” Client Shipping Address Entered in Warehouse Section?
    • If “Mail,” “Bill Postage with Processing” Entered in Comments Section?
  • Sort Matches DP and Lettershop Direction?
  • GEO indicated?
  • Letter sized piece meets size and aspect ratio?
  • Folded Self-Mailer/ Booklet meets USPS folding requirements?
  • Flat meets addressing rules and droop test?
  • Mail panel okay?
    • At least 2x4?
    • No more than 10% screen?
    • Position okay?
    • First class has return address on front or back?
    • Std or Std NP has a return address on the front or back if using a corporate permit.
    • Return address matches client NP# address and CRID address?

Data Processing Tab

  • Work Order Direction to DP reviewed?
    • Do files have multiple tabs?
    • Are there codes to include?
    • Extra Columns not needing loading?
    • # of files to load indicated?
  • List review complete?
    • Are there multiple files?
    • They will all be CASSED?
    • Do they need NCOA? Who will it be returned to?
    • PAF included or on file? Expiration Date?
    • Movecomply included if NCOA not needed?
  • Weight on work order?
  • Mail Foreign indicated?
  • Postage office for mail entry (audit) indicated on W.O.?
  • Presort numbers for Lake System match indicated on W.O. if needed and locations marked on the job samples?
  • Dedupe needed?
  • Make selects or use entire list?
    • Do selects need to be sublisted?
    • WO to sublist to indicated?
  • Does “Output to X” match Postage and Lettershop direction?
    • OEL, T&P, SEQ, BC, COMMINGLE
  • Does a final mail file need to be saved?
  • Are there multiple versions of letter or reply?
    • Does DP know how to code them?
  • NDC/ SCF Processing included?

Lettershop and Fulfillment Tabs

  • Tabs drawn on Self Mailers/ Booklet sample and indicated on work order?
  • Made appropriate selection under ‘Inkjet Proofs’?
  • Does addressing match Output to X and postage tabs?
  • Does postage being applied match Postage tab?
    • Does a stamp order form need to be emailed
  • Have service codes for all actions?
  • Are the # of inserts indicated correct? Has the package sequence been entered in comments section?
    • If not standard package, has Production approved sequence of package?
  • Any special proofing requirements for Production listed and correct?

Print Tab

  • Special order paper or materials needed?
  • Has Print Dept been informed?
  • Will Print Dept need to make any edits to files?
    • Prepress included for edits taking 15 min or less?
    • If more than 15 min, submit Client Services WO
  • Folding Direction included for all necessary pieces?
  • If printing set qty, has this information been written on the WO?

Foreign Tab

  • Set Up Foreign direction included?
  • Is processing same as domestic or will there be additional work?
    • Do envelopes need to be supplied?
    • Do we need to print labels? How should they be formatted?
  • Postage for Canadian & Other Foreign correct?
  • Are we applying stamps? Has service code been included?
  • If foreign weight is different than domestic, has this weight been written on the foreign work order?

Inventory and Warehouse Tabs

  • Has inventory been received into work order?
    • Verified quantities received verses mail quantity?
    • If short, included instructions or contacted printer or client?
  • Is direction for warehouse handling of residuals correct?
    • If return, is the shipping address in the Warehouse comments section? Street address, not P.O. Box?
    • Have you input how many pieces, folded or flat, stuffed, addressed, etc. in service code?
  • Deliver to NDC/SCF included?

Work Order Review

  • Quantity correct on all activities?
  • Is Security Policy required?
    • Have you added it to all sections?
  • Separate work orders for Foreign and Printing if needed?
    • Is the foreign work order in a big job bag with consolidated?
    • Is print on its own?

Sample

  • Sample needed for job bag?
    • Does postage match work order?
    • Is weight on the sample, with math written out?
    • Is there a seq# or T&P needed on outer envelope?
      • Is there at least ½” clearance?
    • Is the sequence on each variable piece in the package?
    • Are the pieces numbered?
    • Are folding samples needed for Print & in job bag?
